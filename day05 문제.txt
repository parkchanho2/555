1. 어느 달팽이는 낮에는 3cm 올라가고 밤에는 2.5cm 내려온다고 할 때, 달팽이가 13cm의 나무 꼭대기에 올라가려면
며칠이 걸리는지 구하시오



























1.풀이
public class Quiz01 {
	public static void main(String[] args) {
		int day = 0;
		double snail = 0.0;
		
		while(true) {
			day ++;
			snail += 3.0;
			
			if(snail >= 13.0) {
				break;
			}else {
				snail = snail - 2.5;
			}
		}
		System.out.println(day + "일");
		
	}
}
